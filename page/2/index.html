【2019】兼职辅导员工作总结

发表于 2019-12-12 更新于 2023-04-13

2019年，匆匆地结束了一年的支教时光，离开乌江中学的三尺讲台的时候突然觉得能再也没机会做老师。承蒙学校和学院领导老师们的信任，回到海大的我加入到了兼职辅导员的大家庭里，这又是完全不同的一种体验。作为一名支教老师，我需要做的是孩子们健康快乐地成长，告诉他们最简单的常识，而作为一名辅导员，我需要和我的学生一起成长，去认识世界的样子，培养自己并引导他们成为更加优秀的人。作为一名年轻的兼职辅导员，经历了许多难忘的事，增长了不少知识，积累了丰富的管理经验，比较圆满地完成了自己所承担的各项工作和任务，现将辅导员工作总结如下：

一、不忘初心、强化学习意识，不断提升学生素质
辅导员要做好学生成长的引路人，就需要在日常学生教育管理中强化学习意识，不断提升学生的综合素质。针对学业警示问题，分析问题学生的成因，开展针对性帮扶工作，注重利用家庭的力量、班级集体自习和专业相关指导对问题学生引导，加强学业监督和特殊关照。针对所带班级实际情况组织就业指导培训会，邀请校外就业导师有针对性地向同学们介绍了目前房地产行业就业形势与政策。在辅导员工作中，要密切联系学生生活实际，准确把握学生思想脉搏，积极回应学生思想关切，帮助学生解疑释惑。

二、统筹兼管、内外联动，加强动态跟踪管理
利用微信群和网络媒介手段对学生进行思想教育宣传以及安全教育，定期不定期地推送一些最新材料，对学生进行动态跟踪管理。在已有管理经验的基础上，不定期地深入班级、深入宿舍，和学生面对面交流100余人次，多方面全方位和学生家长沟通60余小时，家校联动实现了学生的全面管理。本学期突发的带状疱疹和疑似肺结核等事件，第一时间发现问题、掌握动态、解决问题，同时做好密切接触者症状监测和宿舍的通风消毒工作，真正做到安全稳定零风险。

三、牢记使命、思政育人，抓常抓细做好服务
学生管理中没有小事，小到宿舍卫生，大到奖助学金评定，都需要花时间和精力去创新，抓常抓细做好管理服务工作。重点关注所带班级中学业警示同学、学业困难学生、有心理问题倾向的学生和部分涉及宗教信仰学生的近况，实现学生思想工作的全程管理、充分利用信息网络、主题班会、谈心交流等方式，形成“心贴心、面对面、键敲键、网连网”的思想教育工作开展模式，达到指引渗透融入的思想教育目的。

作为辅导员不能单纯追求良心上的认可，不能只停留在认真细心，更应该强调对学生工作深入的思考、细致地落实具体事务，才能在既定的角色中逐渐历练成熟。事实证明，关爱学生、理解学生，主动成为他们中的一员，才能够得到学生的尊重，才能够使学生在原有的基础上不断的前进，才能真正地尽了一个辅导员应尽的职责。最后一句话总结半年来的工作：志之所趋，无远弗届，穷山距海，不能限也。

选择一种不一样的人生

发表于 2019-06-12 更新于 2023-04-13

2018年7月19日23点，跨越两千多公里到达贵州，正式开始了我为期一年的支教生活。
2018年8月23日10点，在志愿者的陪伴下到达播州区乌江中学，这将是我支教故事的起点。

大家好，几个月后我将是一名工程学院机械工程专业的研一新生，而现在，我是一名在大山深处耕耘三尺讲台的支教老师。写这段文字的时间，是我在黔支教生活的第295天，也是到乌江中学的第260天。不知不觉间一年的支教生活马上就要结束了，不知道该怎样用文字来表达过去的种种感受，或许当自己的经历慢慢地变成一段段回忆，带上自己心中曾经的失意、开心、挫败、幸福的种种，便都成为了一个个美好的篇章。记得出征仪式上，研支团的前辈曾经告诉我们，“真羡慕你们呀，年纪轻轻就成为了一群有故事的人！”。而今天，我想做一个讲故事的人，来讲一下我和我们的故事。

我和常见常新的乌江中学
有人将支教生活总结为“三苦三乐”。所谓三苦，即生活的艰苦、对家的思念之苦、心有余而力不足的困苦。但是，在支教生活中获得的快乐却是更高层次的快乐，那就是被尊重的快乐、被需要的快乐、被关心的快乐。
关于乌江中学，我的第一印象仅停留在一所建在山顶的乡村中学，当然这一印象在这短暂的一年中还是发生了巨大的改变。初到乌中，你会发现原来西部真的不是那么好适应的：贵州人的饮食习惯是辣不欢的，所以单是饮食这一关就难倒了我这个“山东大汉”，加上狭窄潮湿的宿舍环境、数不清的奇怪蚊虫、一周七天阴雨的天气，生活着实不易呀；其次，虽然从高中便开始住校，但是当你一个人背起行囊来到几千里外的大山深处，想家依旧是个永不过期的话题，有时候即使再无微不至的关怀也比不上家里妈妈的一顿晚餐；最重要的一点，大学毕业后带着自己的理想与情怀来到西部，在经历了千难万险之后，猛然间发现现实似乎没有丝毫的改变，深深的无力感会迅速地包围着满满负能量的你，这时你会感受到真正的绝望不是失败，而是没有成功的希望。
听我讲完，或许你会感觉我对乌江中学充满不好的感受，但事实却正好相反，当孩子们用期待的眼神望着你写下一个个公式，当你走在校园中听到的一声声老师好，当你感冒生病时学生们一遍遍地问你有没有吃药看医生，当有一天他们可以走出大山看见外面的世界时，你会发现乌江中学这所拥有五十多年历史的乡村中学更像是一位青年人，常见常新，总会带给你不一样的惊喜。正是一代代人用新的面孔描绘出多美的乌中，也正是这位青年人带着大山里的孩子一步步地走向外面的世界。
我和常见常新的乌江中学，偶然相遇，必然离别，割舍不断的是我的青春与这片热土的血肉相连。

我和文武双全的队友们
一年不长的支教时光，从另一个角度来讲更像是一群人的青春旅程，而说到我的队友们，一年的时间足够我们收集互黑的笑料。本就是不大的年纪，来到陌生的土地上，能够想象的到“骚操作”她们从不放过。周末食堂没有饭菜，我们便开始自己做饭，于是便经历了“学习做饭两步法——从开始到放弃”，看看浓烟滚滚的铁锅有时候白粥也挺好喝的，但是她们绝不放弃，白粥、黑米粥、薏仁米粥、红枣枸杞粥、银耳粥···是的，能够想象到的粥她们都进行了尝试，毕竟这是最简单、却可以填饱肚子的方式。当然这也只是我们生活的一个小小片段，凌晨一两点钟的办公室里可能她们依旧在紧张地备课；重大志愿服务活动时看她们自信潇洒地解说；文艺汇演的台下你会看到她们跑前跑后的样子； 才艺比赛的舞台上她们强忍紧张的僵硬笑脸；英语考试没有听力时晚自习后对着录音笔傻傻说话的她们，如果非要用一个词形容我的队友，文武双全是再合适不过的。感谢这种种的经历，感谢我与这群优秀的队友一起酸甜苦辣的日子。
其实对我而言，参加研究生支教团是一段充满考验的愉快经历。在年轻的时候，有机会在这些神奇美丽的地方工作生活一段时间，本身就是一件幸事，而和志同道合的人一起奋斗，进而成为一辈子的朋友，岂不更是人生一大乐事！

我和这群可爱可恨的孩子们
只有真正走上讲台，才会深深地体会到老师是个走心的工作。因为你每时每刻都要有足够的耐心和充足的精力，以应对他们各种奇奇怪怪的问题，尤其是当你面对一群又爱又恨的孩子时。
第一次真正意义上站在讲台上，面对一群十三四岁的孩子，爱恨交织是我最深的体验。他们的小脑袋瓜里充满了成年人理解不了的世界，有时候他会问你宇宙大爆炸是怎样产生了我们这个物质世界，有时候他又会跟你讨论灭霸的战力到底有多强，前面一秒他还在为浮力的产生原因冥思苦想，后一秒可能他的想法又飞到了食堂阿姨是否会多给打一块红烧肉。当然，课堂上交头接耳、互传纸条的讨厌行为也是他们的家常便饭，他们就像是一群遗落人间古灵精怪的“小恶魔”，带给你开心的同时也能分分钟让你火冒三丈。
经过长期的实践，你会渐渐找到作为老师的那种快乐和成就。那种成就是当你一笔一画地写在黑板上，孩子们望着黑板渴望的眼神；那些快乐就是在批改作业本时出现的一个又一个的“对勾”中。渐渐地也便明白了，支教的意义不仅是教给孩子们知识，更重要的是带给孩子们一个积极的人生态度，让他们勇于做梦，陪伴他们成长。

写在最后的话
时光不停，故事不断，我和我们的故事在一直继续中。
青年，归根结底要活在自己喜欢的事业追求中，要活在别人对你发自内心的尊重和认同中，其实大多数时候，感人的未必是事迹，而是我们从相似的故事里看到自己曾经的热情与真挚，愿每一位青年人都能保持初心。

念之再三，铭之肺腑

发表于 2018-06-12 更新于 2023-04-13

很高兴在这里进行我今天的演讲，今天的演讲主题是2016年政府工作报告中的一句话，相信大家或多或少都看过今年李克强总理所做的政府工作报告，报告全文19574个字，内容方面，从稳经济、调结构到大众创业、万众创新和“互联网+”，可以说涵盖人民生活的方方面面，而我今天则断章取义的选取其中一句话谈谈我的感受。
一年一度的政府工作报告从来都是社会关注的焦点，而民生也是近些年来关注度极高的话题，在今年总理的政府工作报告中谈到民生问题时说了这样一句话，“为政之道，民生为本，我们要念之再三，铭之肺腑，多谋民生之利，多解民生之忧。”虽然我们不用像总理一样为整个国家的人民负责，但我们却也肩负着一个班集体的责任，在干训班中我曾经学到过这样一句话，“学生干部的作用就是将党和国家的大道理变成同学们喜闻乐见的小道理。”那么，我就从一名刚刚做了半年班长的角度讲讲“念之再三，铭之肺腑”的理解。
大家是否还记得班干部竞选的时候你在大家面前说过什么，你是否还能体会当你说完你的竞选词时的紧张心情，或许那已经太遥远了，远到连你自己都不清楚你到底完成了多少。在这里我想分享一下我的记忆，每个人都有自己的梦想，我认为一个班集体也不例外，我说过我希望我们的班级是一个有爱的大家庭，我们来自祖国的四面八方，我们拥有不同的人生经历和喜爱，但是我们真心爱这个集体。在我们支书生日的时候她发了一条说说，而说说的内容中有一句话特别让我感动，“在心里面，家人第一位，机械第二，不知道下一次还会不会站在那个台上继续竞选，但确实也难放下心中的牵挂，以后的选择留给以后吧”。对于班级，我们有梦想，我们不忘初心，风雨兼程。
另外一点，记得看过这样一句话，梦想是一种诅咒。功夫熊猫里阿宝面对父亲对他梦境的询问，他弱弱的回答，“嗯，是关于面条的！”梦想的代价有时候是巨大的，大家都知道我们不能总期待家人为我们的梦想买单，那么大家有没有想过班集体的梦想又该谁来买单呢？小的时候我们愿意为了一只蝴蝶追一整个下午，高中的时候愿意为一道压轴题浪费一节自习课，而现在我们却不愿意用过十几分钟的时间去宿舍和自己的同学聊聊，其实最初的我们并不是这样的，只是不知道在什么时候开始我们已经忘记了坚持。当你走的太远，以至于找不到路的时候，回头看一下来时的路也便找到自己的目标，班级里我们有坚持，我们心若向阳，无畏悲伤。
回到我演讲最初的一句话，念之再三，铭之肺腑。我们再三考虑的是什么，我们铭记在身心的又是什么，无论是做人还是还是做事，对生活，要有梦想，不忘初心，风雨兼程，对于梦想，要有坚持，心若向阳，无畏悲伤。

致谢青春，再见海大

发表于 2018-05-20 更新于 2023-04-19

蒲月琴岛，鲜荷摇碧，老苔积青。竭思苦虑拙作良久，不求一鸣惊人，惟图学业功成。虽理浅拙，然吾兢业之心，鉴于日月。值吾文稿近余之际，忆往昔，四载春秋，白驹过隙，虽所学不精，亦觉受益良多。

鄙非感伤溅泪之士，而今回首，亦未免五味杂陈。癸巳仲夏，初入学塾，载怯载欣，志硕学宏才，欲求索济国。然今日三省吾身，虽才本疏浅，亦不负四年之志。东方学府，书山浩浩，学海淼淼。常秉烛求索兮，恐有负重托。学在海大，拥旖旎于身畔，曾五子顶揽风，亦八关山读月，锦瑟年华，春花秋月共赏。每每念及海大时光，不负青春与梦想。

行文至此，宜拜谢众人，然欲语忘言。一谢父母，身体发肤授之父母，十月怀胎恩情深，卧冰求鲤报恩轻。二谢穆师，儒雅谦和，睿智博学，师生融融，此诚吾生之大幸也。三谢师兄文声、建刚，亦友亦师，予余之助，不胜感激。四谢同窗，四五同窗，风华正茂，青春岁月，各具豪肠。或歌或舞，莫诉离殇。五谢吾之坚持，虽不才愚钝，然持之以恒。所学艰深，吾终有一二之所得。

恩情深而笔墨浅，情谊重而言语拙。情长意深，一语难诉，惟寄情于心间。不忘父母恩，铭记恩师情，常忆同窗谊，无语可概此深情，再拜谢！

我和我们

发表于 2017-06-15 更新于 2023-04-19

很久之前就有想法说写一些关于我在学生会这三年的文字，却总是不知道该怎样提笔。前几天准备主席团换届的时候，一仨突然提起去年桐哥留给我们的《一只老狗的送白》，重新读来依旧感觉那段时光的珍贵和怀念。我没有桐哥的细腻感情和飞扬文采，自己也觉得不适合走煽情路线，所以也就不去做一篇告白推送，就简简单单的写写自己的感受吧
夜已经深了，这三年也习惯了深夜的挑灯夜战。这几天感冒愈加严重，整个人恍恍惚惚，也就是这样结束了我与学生会的三年爱情陪跑。我不是一个特别感性的人，本来想写的只是《我们的最后一次晚点名算是给一年的生活和陪伴我的人一个总结，思虑再三还是想说说我的三年。

一
2014年秋，我第一次走进大学、第一次进入秘书处、第一次和一群朋友开始我的大学生活。
还记得那个时候的我和我们都还是简单的样子:阿姨(娜姐)是最无微不至地照顾我们的大姐姐，元哥一致让我觉得古灵精怪(形容男生怪怪的)，有的时候特别正经，有的傻里傻气，颖姐从来都是美美哒，以及联想到的曲教官，楠哥也是颠覆我对帅气国防生印象的罪魁祸首，舟舟最不像一个部长，挺像一个需要我们照顾的小妹妹。当然，也还有一起搬桌子的秘书处宝宝，一群非要自封“后勤部”的秘书处。
现在回想起我的大一，依旧好多场景历历在目。迎新晚会，我们在舞台后面看了一晚上的物资;成立大会后，我们吃着零食看悦姐给我们演唱苦行僧;冷餐会上，我把马毓龙读成了马硫龙;运动会早上，五点钟去打扫场地，日租房里，阿姨用介绍女朋友的方式骗大头吃了一盘焦糊的拔丝地瓜;流清河的晚上，听斌叔弹着吉他唱着歌;最后的聚餐，听楠哥喝醉之后的心灵鸡汤….
阿姨，给我们找个叔叔吧!
元哥，你的高数一定会过的!
楠哥，多喝牛奶多读书，你一定会长高的!
颖姐，你要和曲教官好好哒!
舟舟，做我的女朋友吧!
怀念熬了三个通宵剪出来的视频，怀念那个时候的点点滴滴，不只是喜欢那个时候的我们的，还喜欢那个时候的自己。

二
2015年初夏，我当部长啦!一丝激动，一点紧张，一份踌躇，我们五个人努力撑起秘书的天空。
那个时候的我们动力满满。迎新期间，夏季学期早上到办公室打卡，中午是外卖，尤其是最后的几天几乎都是10点钟才往宿舍赶，迎新的当天我睡眼朦胧地打开办公室门口的时候，桐哥整个人像一个大字躺在一堆赞助袋上。没有机会看到干事们的军训是遗憾的，黄岛的实习也让我错过了干事们的面试复试，至今仍记得特意去买了cmcc的账号，对着照片记你们的名字，在群里冒充小鲜肉和你们聊天。运动会之前，我的胳膊伤了，从医院出来的第三天匆匆地赶回学校早上看到你们都在乖乖地布置场地、记录成绩真的心安不少。我们的冷餐会，我的胳膊依旧是吊着石膏的，看着你们略显尴尬的主持、努力认真的态度、美美哒PPT，感谢你们的付出。
我和我们，从来都是最好的配合。感谢我的干事们让我这个胖子部长有如此大的成就感，一切都是你们的成功，我只是一个领路的胖子，不管我这个向导怎样，都希望你们的大学可以开心快乐。

四
2017年06月13日，我们终于退休啦。看到新一届的主席团产生突然有种时光飞逝的感觉，新的一学期希望你们加油。在学生会待了三年啦，以后再也不用每周开例会啦，也不用晚睡早起地赶活动，本该高兴的事情还是有点伤感，有一些乱七八糟的话写给自己：
体验了一群朋友一起努力的日子，那我们的青春就不曾荒废。
付出真心，体验过程，结果或许重要，但是我更看重我们。
永远不要让不理智占据你的内心，世界很大，外面很美，我们多走走。
